Программа предназначена для ввода данных о человеке и записи в файл.
При запуске программа демострирует пользователю главное меню:

Введите цифу, соответствующую пункту меню:
1 - Ввод данных.
2 - Прросмотр введенной информации.
3 - Запись введенной информации в файл. Выход.

После ввода "1" программа предложит пользователю ввести ФИО человека.
ФИО должны быть внесены строго по порядку, и содержать только буквы русского алфавита.
Программа проверит введенные данные на количество введенных слов и используемые символы.
В случае неправильного ввода, программа выдаст соответсвующую ошибку и вернется в главное меню.
Если ввод был произведен правильно, программа предложит пользователю ввести номер телефона,
дату рождения, и пол. Данные можно вносить в произвольном порядке.
Формат вносимых данных:
пол - символ латиницей "f" или "m", допускается ввод заглавных букв.
номер телефона - 11 цифр без знаков.
дата рождения - дд.мм.гггг допускается ввод только по заданному шаблону.
В случае, если какой-то из параметров был введен неверно, пользователь получит сооветствующее сообщение.
При правильном вводе, программа распознает введенные данные, создаст объект класса PersonalInfo,
сохранит его в personalInfoList и запишет соответствующие данные в файл, названием которого будет служить введенная фамилия.
После этого программа вернется в главное меню. Введя цифру "2" пользователь имеет возможность просмотреть введенные данные.
Отображаются только корректно введенные записи. При вводе "3" все, корректно введенные данные,
будут сохранены в соответствующие файлы и программа завершит свою работу.
